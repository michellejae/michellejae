hikeKnow

project managed a group project that developed a hiking trail app, ‘HikeKnow’, that gathered weather and trail data on local hikes for the Island of Oahu and recommended daily hikes based off of information collected from three separate API’s. Utilized async-await and node-scheduler to call API’s at various times during the day to keep all data relevant for each hike. frontend was built using the AngularJS framework, SASS and bulma library. backend was built with NodeJS, express and PSQL. redeployed using Digital Ocean, Nginx, LetsEncrypt and PM2 and set up webpack from scratch.
